Executing C++ Code

For those who have installed c++ on their machines and will be using a terminal to run programs.
C++ Programs can be written on any text editor and saved as a file having a file having a .cpp extension.

  • Step 1 Open terminal/Cmd.
  • Step 2 Change the Directory to the one in which you have your program file with .cpp extension

            < Your Directory >

  • Step 3 Now enter the following to compile the file using g++

            g++ filename.cpp -o anyname

  • Step 4 Finally Run the program by

           ./anyname

Loops and Conditionals

Conditional branching and looping is an integral part of any programming language

  • if , if..else , if..else if..else

    The if statement is used to run a part of code only if a particular condition is met.

    The Syntax is as follows

    if(condition)
    {
      execute this
    }

    The if statement can also have an optional else clause

    The Syntax is as follows

    if(condition)
    {
      execute this
    }
    else
    {
      do this
    }

    However if we need to make a choice between more than two cases we can use if..else if..else

    The Syntax is as follows

    if(conditon)
    {
      execute this
    }
    else if(condition2)
    {
      then execute that
    }
    else
    {
      do this
    }

    for(initialization, conditon, increment/decrement)
    {
      code to be repeated
    }

    Initialization initializes the loop varible, executed once.

    condition if true then loop body executed else loop gets terminated.

    increment/decrement updates the value of loop variable and again checks condition
